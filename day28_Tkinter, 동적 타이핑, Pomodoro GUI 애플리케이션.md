## [[Event Driven]]
- 이벤트 구동형
- 콘솔에서 뭔가를 할 때 -> 엔터를 눌러 입력
- GUI -> 본질적으로 리프레싱을 하고 계속 이벤트를 주시해야 함, 실시간 반응
- [[tkinter]]에서 `mainloop()`를 통해 구현

## [[Dynamic Typing]]
- 동적 타이핑
```
if count_sec == 0:      # count_sec -> int
	count_sec = "00"    # count_sec -> str
```

- 변수 안의 콘텐츠를 변경해서 변수의 데이터 형을 변경할 수 있음
